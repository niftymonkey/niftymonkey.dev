# niftymonkey.dev

Terminal-inspired homepage showcasing software projects with a minimalist, developer-focused aesthetic.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Font**: Monospace (JetBrains Mono or Fira Code)

## Design Aesthetic

Dark terminal color scheme with monospace typography, terminal-window styled project cards, cursor blink animations, and terminal prompt aesthetics.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Project Structure

```
/app          - Next.js app router pages and layouts
/docs         - Project documentation and planning
/public       - Static assets
```

## Adding Projects

Projects are configured in a simple TypeScript config file for easy maintenance and expansion.

See `docs/PROJECT_PLAN.md` for detailed implementation plans and design decisions.
